# OpenGL Learning Journey

Welcome to my OpenGL learning repository! This project documents my progress as I explore **Modern OpenGL (3.3+ Core Profile)**.

The goal is to deeply understand the graphics pipeline, shader programming, and real-time rendering techniques by building small, incremental projects from scratch.

## Built With

- **C++**: The core language for performance and control.
- **OpenGL 3.3**: Using the Core Profile for modern cross-platform graphics.
- **GLFW**: For window creation, context management, and input handling.
- **GLAD**: For loading OpenGL function pointers.

## Guided By
This project follows the excellent tutorials from [LearnOpenGL.com](https://learnopengl.com/).
