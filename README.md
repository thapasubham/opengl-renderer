# OpenGL Learning Journey
This repository contains experiments and learning projects with **modern OpenGL**. The goal is to explore the OpenGL graphics pipeline, shaders, and simple animations while building small 3D scenes from scratch.

It documents my progress while learning modern OpenGL using **GLFW** and **GLAD**.  

Each example builds on concepts from [LearnOpenGL](https://learnopengl.com/) and includes my own experiments.
